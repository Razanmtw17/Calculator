# Simple Calculator #

This is a simple calculator application built using C# and Windows Forms.

Features:

Basic arithmetic operations (+, -, *, /)
Number input with decimal point
Clear button to reset the display
Getting Started

Prerequisites:

Visual Studio 2022
Basic understanding of C# and Windows Forms development

Running the Calculator:

Press F5 to build and run the application.
Usage

Enter numbers using the number buttons (0-9).
Click the operator buttons (+, -, *, /) to perform calculations.
Click the decimal point button (.) to include decimals in your numbers.
Click the equals button (=) to display the result of the calculation.
Click the clear button (C) to reset the calculator and start a new calculation.
